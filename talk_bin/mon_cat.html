2 Monoidal categories

Tensor products are essential to defining Hopf algebras, and we will see that a monoidal structure is the categorical generalisation of a tensor product. In the following all structures are taken to be linear over a field kkk of characteristic 000, usually just assume k=Ck=\mathbb {C}k=C.

Given two kkk-vector spaces V,WV,WV,W, the tensor product V⊗WV\otimes WV⊗W is a vector space made up kkk-linear combinations of symbols called “pure tensors” v⊗wv\otimes wv⊗w for v∈V,w∈Wv\in V,w\in Wv∈V,w∈W satisfying:

  • λ(v⊗w)=(λv)⊗w=v⊗(λw)\lambda (v\otimes w)=(\lambda v)\otimes w=v\otimes (\lambda w)λ(v⊗w)=(λv)⊗w=v⊗(λw) for λ∈k\lambda \in kλ∈k

First note that k⊗V≅Vk\otimes V\cong Vk⊗V≅V, which we see via the linear map λ⊗v↦λv\lambda \otimes v\mapsto \lambda vλ⊗v↦λv with inverse v↦1k⊗vv\mapsto 1_ k\otimes vv↦1k​⊗v. Secondly, given linear maps ϕ:V→W,ϕ′:V′→W′\phi : V\xrightarrow {}W, \phi ':V'\xrightarrow {}W'ϕ:V​W,ϕ′:V′​W′ we can define the tensor product linear map as ϕ⊗ϕ′:V⊗V′→W⊗W′, v⊗v′↦ϕ(v)⊗ϕ(v′)\phi \otimes \phi ':V\otimes V'\xrightarrow {}W\otimes W',\ v\otimes v'\mapsto \phi (v)\otimes \phi (v')ϕ⊗ϕ′:V⊗V′​W⊗W′, v⊗v′↦ϕ(v)⊗ϕ(v′). Note we only defined ϕ⊗ϕ′\phi \otimes \phi 'ϕ⊗ϕ′ on pure tensors, however the assumption that ϕ⊗ϕ′\phi \otimes \phi 'ϕ⊗ϕ′ is a linear map is enough to deduce its action on the rest of V⊗V′V\otimes V'V⊗V′.

The reason why tensor products are so useful is that they satisfy the following universal property: for every bilinear map f:V×W→Zf:V\times W\xrightarrow {}Zf:V×W​Z there exists a unique linear map fˉ:V⊗W→Z\bar{f}:V\otimes W\xrightarrow {}Zfˉ​:V⊗W​Z such that fˉ(v⊗w)=f(v,w)\bar{f}(v\otimes w)=f(v,w)fˉ​(v⊗w)=f(v,w). So, in effect, we can turn bilinear maps into linear map. We put this to use with an alternative definition for an algebra:

There are several advantages to Definition 2, the first being that the axioms that define an algebra get reformulated as commuting diagrams. We will also express the axioms for a Hopf algebra as commuting diagrams and this helps a lot to simplify its definition. The second advantage is that by replacing bilinear maps with linear maps, we allow for a more categorical definition of an algebra. In particular, we can view an algebra as an object AAA in Vect\textbf{Vect}Vect, the category of kkk-vector spaces, with morphisms m:A⊗A→Am:A\otimes A\xrightarrow {}Am:A⊗A​A and η:k→A\eta :k\xrightarrow {}Aη:k​A satisfying the commuting diagrams above. A crucial part of the category Vect\textbf{Vect}Vect that allows for the construction of algebras inside it is that is has notion of tensor product. Therefore we might guess that algebras could be defined in other categories so long as they have something that plays the role of the tensor product:

2.1 A detour into the Coherence theorem

It is not obvious why we should want the first diagram above to commute (known as the pentagon condition). Note that given an ordered sequence of objects X1,…,XnX_1,\dots ,X_ nX1​,…,Xn​ in CCC we can take their tensor product, but depending on the order in which take tensor products we get different bracketings on X1⊗⋯⊗XnX_1\otimes \dots \otimes X_ nX1​⊗⋯⊗Xn​, giving distinct objects of CCC. We wish to identify the different bracketings so that we can speak of X1⊗⋯⊗XnX_1\otimes \dots \otimes X_ nX1​⊗⋯⊗Xn​ unambiguously.

When n=3n=3n=3, we know by Φ\Phi Φ the two ways of bracketing give isomorphic objects X1⊗(X2⊗X3)≅(X1⊗X2)⊗X3X_1\otimes (X_2\otimes X_3)\cong (X_1\otimes X_2)\otimes X_3X1​⊗(X2​⊗X3​)≅(X1​⊗X2​)⊗X3​. For general n≥3n\geq 3n≥3 it can then shown there is an isomorphism between any two bracketings of X1⊗⋯⊗XnX_1\otimes \dots \otimes X_ nX1​⊗⋯⊗Xn​ by repeated applications of Φ\Phi Φ. There is a beautiful way to see this using graphs theory. We identifying each bracketing of X1⊗⋯⊗XnX_1\otimes \dots \otimes X_ nX1​⊗⋯⊗Xn​ with a vertex, and connect vertices by an edge if they are related by an application of Φ\Phi Φ. The result is a convex polytope known as an “associahedron". The associahedron in the case n=4n=4n=4 is actually depicted in the pentagon diagram above, i.e. it is a pentagon. The case n=5n=5n=5 is depicted below.

We see that any two bracketings can be related by applications of Φ\Phi Φ as a result of the fact that there is a path between each pair of vertices on the associahedron, i.e. it is a connected graph.

So we have that the different bracketings give isomorphic objects, however to unambiguously identify them we need to ensure they are related by unique isomorphisms. Looking at the associahedron we see there can be several different paths between any two vertices, corresponding to different isomorphisms between two bracketings of X1⊗⋯⊗XnX_1\otimes \dots \otimes X_ nX1​⊗⋯⊗Xn​. On the pentagon diagram (i.e. when n=4n=4n=4) we essentially have two possible routes between any pair of vertices. The pentagon condition equates the two routes between ((V⊗W)⊗Z)⊗U((V\otimes W)\otimes Z)\otimes U((V⊗W)⊗Z)⊗U and V⊗(W⊗(Z⊗U))V\otimes (W\otimes (Z\otimes U))V⊗(W⊗(Z⊗U)), but with a little thought one can see that this means the routes between any pair of vertices on the pentagon diagram are equal. The content of the “Coherence theorem" is that this condition is sufficient to deduce that all bracketings of X1⊗⋯⊗XnX_1\otimes \dots \otimes X_ nX1​⊗⋯⊗Xn​ are related by a unique isomorphism for all n≥3n\geq 3n≥3, and so they can be canonically identified with each other.

2.2 Examples

The obvious example of a monoidal category is Vect\textbf{Vect}Vect, the category of kkk-vector spaces. It can be checked that the tensor product as defined at the start satisfies all the conditions to make Vect\textbf{Vect}Vect a monoidal category, where we take the unit to be the field, i.e. 1:=k1:=k1:=k. However a more interesting example comes from representation theory of groups.

Representations of groups

Recall for a group GGG and vector space VVV, a group representation is a group homomorphism ρ:G→GL(V)\rho :G\xrightarrow {}\text{GL}(V)ρ:G​GL(V). GL(V)\text{GL}(V)GL(V) is the set of invertible linear maps V→VV\xrightarrow {}VV​V, and under a choice of basis {v1,…,vn}\{ v_1,\dots ,v_ n\} {v1​,…,vn​} for VVV this set can be identified with GLn(k)\text{GL}_ n(k)GLn​(k), the n×nn\times nn×n-invertible matrices over kkk. So the map ρ\rho ρ sends elements of our group GGG to linear maps (or matrices) on VVV in a way that respects the group structure.

Indeed it can be checked this forms an algebra, so in particular it is also a ring, so we can consider modules over kGkGkG. Recall a (left) RRR-module is an abelian group VVV (for instance a kkk-vector space) with an action ⊳:R×V→V\rhd :R\times V\xrightarrow {}V⊳:R×V​V satisfying (r⋅s)⊳v=r⊳(s⊳v)(r\cdot s)\rhd v=r\rhd (s\rhd v)(r⋅s)⊳v=r⊳(s⊳v) and 1R⊳v=v1_ R\rhd v=v1R​⊳v=v.

The following result is very important in that it shows representation theory is just a part of module theory. Let kGkGkG-Mod\text{Mod}Mod denote the category of left kGkGkG-modules with morphisms being kGkGkG-module homomorphisms.

Proposition 2.6

There is an isomorphism between the categories Rep(G)\text{Rep}(G)Rep(G) and kGkGkG-Mod\text{Mod}Mod.

Proof
For every group representation ρ:G→GL(V)\rho :G\xrightarrow {}\text{GL}(V)ρ:G​GL(V) there is a left kGkGkG-module structure on VVV defined via the following action: (∑gλgg)⊳v:=∑gλgρ(g)(v)(\sum _ g \lambda _ g g)\rhd v:= \sum _ g \lambda _ g \rho (g)(v)(g∑​λg​g)⊳v:=g∑​λg​ρ(g)(v) It is very quick to check this satisfies axioms to be a module. Conversely, given a kGkGkG-module VVV, define ρ:G→GL(V)\rho :G\xrightarrow {}\text{GL}(V)ρ:G​GL(V) as ρ(g)(v):=g⊳v\rho (g)(v):=g\rhd vρ(g)(v):=g⊳v. And again the module axioms mean that this is indeed a representation. We have implicitly defined functors between these two categories (although not explained how the functors act on morphisms). It remains to check that the action under one of the functors and then the other leaves the object unchanged, making these functors mutually inverse. □
We see subrepresentations are equivalent to kGkGkG-submodules, and irreducibility of a representation is equivalent to the corresponding kGkGkG-module being simple. For more this see Section 3.2 of Sengupta [ 3 ] .
